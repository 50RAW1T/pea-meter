🔗 Auto-sync จาก Google Sheet
วาง URL ที่ได้จาก Apps Script แล้วกด บันทึก — ระบบจะดึงข้อมูลสดทุกครั้งที่เปิดเว็บ
📋 วิธีติดตั้ง (ทำครั้งเดียว 5 ขั้นตอน)
1 เปิด Google Sheet → เมนูบน ส่วนขยาย → Apps Script
2 ลบโค้ดเดิมทิ้ง → วางโค้ดจากไฟล์ Code.gs ที่ได้รับ → กด 💾 บันทึก
3 กด Deploy → New deployment → ประเภท Web app
4 ตั้งค่า Execute as: Me · Who has access: Anyone → กด Deploy → อนุญาต permission
5 คัดลอก Web app URL → วางในช่องด้านบน → กด บันทึก
หมายเหตุ
• URL จะบันทึกใน browser นี้ ไม่ต้องตั้งค่าซ้ำ
• ถ้าโหลดสดไม่ได้ ระบบใช้ข้อมูล snapshot แทนอัตโนมัติ
• ถ้าอัปเดต Apps Script ต้อง Deploy เวอร์ชันใหม่ทุกครั้ง
